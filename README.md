## Bill of Materials
- 1x Seeed XIAO RP2040
- 4x MX-Style switches
- 4x Blank DSA keycaps
- 1x 0.91" OLED display
- 2x SK6812 MINI-E LEDs
- 4x M3x5mx4mm heatset inserts
- 4x M3x16mm screws
